Detailed Description

Definitions for CLI argument parsing, flag enumeration, and command-line interface entry point.

Contains the argument structure for storing parsed options, the flags enumeration for operation selection, and the main CLI function signature. Used across all modules to share parsed configuration and validation state.

enum class flags
strong

Enumeration of available command-line operations and parser states.

Ensures mutual exclusivity of image processing commands. The parser switches to the corresponding enum value when a primary operation flag is encountered.

Parses command-line arguments and populates the argument structure.

Processes short and long options using getopt_long, validates operation exclusivity, checks for missing required parameters, handles ignored/extra arguments, enforces coordinate ordering rules, and sets up input/output file paths. Stops execution and returns an error code on any validation failure.

Parameters
[in]argcNumber of command-line arguments passed to main().
[in]argvArray of command-line argument strings.
[out]argumentsReference to the structure that will be filled with parsed data.
Returns
error_marker_t ERR_OK on successful parsing and validation, or a specific error code (ERR_INCORRECTARG, ERR_FEWARGS, ERR_EXTRARGS, etc.) on failure.
