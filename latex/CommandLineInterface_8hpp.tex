\doxysubsection{Detailed Description}
Definitions for CLI argument parsing, flag enumeration, and command-\/line interface entry point. 

Contains the {\ttfamily argument} structure for storing parsed options, the {\ttfamily flags} enumeration for operation selection, and the main {\ttfamily CLI} function signature. Used across all modules to share parsed configuration and validation state.

Parses command-\/line arguments and populates the argument structure. 

Processes short and long options using getopt\+\_\+long, validates operation exclusivity, checks for missing required parameters, handles ignored/extra arguments, enforces coordinate ordering rules, and sets up input/output file paths. Stops execution and returns an error code on any validation failure.
