\doxysubsection{Detailed Description}
Core class for managing, processing, and persisting 24-\/bit BMP images. 

Provides a complete abstraction over the BMP file format, handling binary headers, pixel data storage, and common image manipulation operations such as mirroring, copying, color replacement, and grid splitting.
