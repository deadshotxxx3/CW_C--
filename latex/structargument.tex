\doxysubsection{Detailed Description}
Container for all parsed command-\/line arguments and validation states. 

Holds file paths, operation parameters, coordinate/color data, and boolean tracking flags ({\ttfamily has\+\_\+\texorpdfstring{$\ast$}{*}}) to distinguish user-\/provided values from default-\/initialized ones. Populated entirely by CLI().
