\doxysubsection{Detailed Description}
Fundamental data types, binary structures, and error codes for BMP image processing. 

Provides packed BMP header definitions (\doxylink{structBitmapFileHeader}{Bitmap\+File\+Header}, \doxylink{structBitmapInfoHeader}{Bitmap\+Info\+Header}), color representation (\doxylink{structPixel}{Pixel}), geometric coordinates (\doxylink{structCoordinate}{Coordinate}), and the error\+\_\+marker\+\_\+t enumeration. Ensures strict memory layout compliance with the BMP v3 specification via \#pragma pack directives and fixed-\/width integer types. Serves as the core type system shared across CLI parsing, I/O operations, and image manipulation modules.
