\doxysubsection{Detailed Description}
In-\/memory representation of a BMP image with manipulation capabilities. 

Stores file and info headers alongside a 2D pixel matrix. The matrix follows the standard BMP bottom-\/up orientation, where the first row index corresponds to the bottom of the image.
